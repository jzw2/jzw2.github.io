Welcome to my blog/website. 

Why did I make this? Previously, my website looked like this. 


![My Amazing Website](/images/old_website.jpeg)


Which already looks great? Why did I change it? 

Well, I recently became aware that I need money, and to get money, I need a job. And employers hate fun, so I've decided to remove all the fun from my website.

The previous website was also using purescript, a compile to javascript functional programming language that is basically Haskell. 
Unfortunately, employers are not as impressed. They actually become more disinterested if you start talking about monads. Even though function programming is the most beautiful type of programming, all they see is a website that looks like something a 10 year old in 2005 would make. 

So I decided to switch to a format that looks a lot more impressive to anyone unfamiliar with the details, but in reality, only took me about 10 minutes, because I just stole a template and changed the names.

Hopefully I will be able to post a bit more about my life/projects I've been working on.
